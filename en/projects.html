To restart a system

This PowerShell script automates scheduling restarts for network computers. It includes functions to manage scheduled tasks and a main function (PlanNeustart) that allows scheduling restarts by specifying date, hour, and minute. Additionally, the script adjusts the console window and provides a text-based user interface. It is particularly useful for system administrators in networked environments.
